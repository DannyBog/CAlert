# CAlert
Simple utility that sits in system tray, monitors Home Front Command alerts and notifies with Windows toast notifications when they're recieved.

![CAlert](https://github.com/DannyBog/CAlert/assets/22415309/4b1eb113-1359-4972-b867-f6c2743b47c5)

# Download
You can get the latest build as a zip archive here: [CAlert.zip](https://github.com/DannyBog/CAlert/files/14009984/CAlert.zip)

# Building
* Install [Visual Studio 2022](https://visualstudio.microsoft.com/vs/)
* Open `x64 Native Tools Command Prompt for VS 2022`
* Run `build.bat`
